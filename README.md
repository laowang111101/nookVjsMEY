# 图书零售监测系统 python945

## 项目概述

图书零售监测系统是一个基于Echarts和Python开发的平台。该系统通过整合数据分析和用户服务，为用户提供了一个便捷的图书查询和购买环境。

## 技术栈

- **后端**: Python Django
- **数据库**: MySQL
- **前端图表**: Echarts

## 功能模块

- **用户模块**
  - 用户注册与登录
- **信息查询**
  - 图书查询
  - 论坛查看
  - 公告浏览
- **购物车管理**
- **用户权限管理**
  - 管理员
  - 普通用户

## 系统角色

系统设定了以下两种角色：

- **管理员**: 负责系统管理和内容发布等操作。
- **普通用户**: 可以查询图书、论坛、公告等，并进行购物操作。

## 运行环境

- Python环境搭建（推荐使用虚拟环境）
- MySQL数据库服务

## 部署步骤

1. 安装Python及Django框架
2. 配置MySQL数据库
3. 导入系统所需的数据表结构及数据
4. 运行系统

## 目录结构

```
项目根目录/
│
├── app/               # 应用代码目录
│   ├── models.py      # 数据模型

│

│
├── manage.py          # 管理脚本
└── ...
```

## 核心亮点

- **功能丰富**: 系统整合了用户注册、图书查询、论坛讨论、购物车管理等多种功能。
- **用户权限细分**: 精准区分用户角色，为不同角色赋予相应权限。
- **数据可视化**: 使用Echarts进行数据展示，使数据分析更为直观。
